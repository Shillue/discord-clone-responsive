## 📺 Projeto: Clone Responsivo da Landing Page do Discord

Este projeto foi desenvolvido como parte do desafio **"Construindo um Layout Responsivo para o Site do Discord com CSS"** da DIO, com foco em aplicar conceitos de **HTML**, **CSS** e, principalmente, **responsividade**.

---

## 🎯 Objetivo

Reproduzir a página inicial do site do Discord com fidelidade ao layout proposto, mantendo espaço para criatividade e identidade própria. O foco principal é a criação de um **layout responsivo e bem estruturado**, adaptável a diferentes tamanhos de tela. O projeto também visa reforçar na prática os conhecimentos de **HTML semântico** e **estilização com CSS**, especialmente com o uso do **Flexbox**.

---

## 🖥️ Sobre o Projeto

Este é um clone fiel da **landing page oficial do Discord**, desenvolvido com **HTML e CSS puros**. O layout se adapta automaticamente a telas menores, garantindo uma boa experiência tanto no **mobile** quanto no **desktop**.

🎨 O layout de referência foi disponibilizado pela DIO via Figma: 🔗[Link do Figma - Desafio de Responsividade](https://www.figma.com/design/NRBYrG5d4DSzObv7dpTqoM/Desafio-Responsividade---DIO?node-id=0-1&p=f&t=hiJujufuNxy5JGAY-0)

---

## 📘 Funcionalidades & Conteúdos

- ✅ Cabeçalho com imagem de fundo, título e botões de ação
- ✅ Seções com imagens e textos explicativos sobre o Discord
- ✅ Layout organizado em **Flexbox**
- ✅ **Responsividade** para diferentes tamanhos de tela via `responsive.css`
- ✅ Rodapé com logo e créditos

---


## 🏷️ Tecnologias Utilizadas

**HTML5:**
- Estrutura semântica com `header`, `main`, `section`, `footer`
- Uso de `figure`, `img`, `h1`, `h2`, `p`, `button`, `a`

**CSS3:**
- Layout com **Flexbox**
- Fontes do Google Fonts (`Open Sans`, `Luckiest Guy`)
- Responsividade com **Media Queries**, com foco principal em telas com largura até `860px`. 
- Estilização separada entre `style.css`, `responsive.css` e `reset.css`.

---

## 🧠 Aprendizados

- Aplicação prática de layout com **Flexbox**
- Separação clara entre CSS base e CSS responsivo
- Controle de imagens, tamanhos e alinhamento responsivo
- Implementação de estilos consistentes com design profissional (via Figma)
- Melhoria nas práticas de organização e semântica HTML
